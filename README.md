## Olá!

Sou um Técnico em Desenvolvimento de Sistemas formado no SENAI/SC. Tenho grande interesse em desenvolvimento web e sistemas operacionais, e venho consolidando meus estudos por meio de projetos práticos e outras experiências. 

Ao longo da minha formação, aprendi lógica de programação, fundamentos de eletroeletrônica básica, programação de aplicativos, banco de dados, versionamento com Git e GitHub, desenvolvimento web e IoT, além de modelagem de sistemas com foco em análise de requisitos, documentação técnica.

Me considero uma pessoa muito curiosa, dedicada e comprometida com o meu crescimento profissional. Estou aberto a novos desafios e conexões que possam me ajudar a evoluir como desenvolvedor e construir uma carreira sólida.
